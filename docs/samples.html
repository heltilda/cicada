Cicada sample scripts

The sample scripts include highlights from the two '.cicada' script files that come with the download. Additionally, the help file is a rich source of examples, in most places accompanied by blow-by-blow explanations of what is going on.

  • from user.cicada:
    • the indispensable sprint() routine
    • a function that runs a constructor
    • sort(), a newish addition to user.cicada

  • a somewhat antiquated disassembler makes a nice example.

  • Finally, there is a more extended example in the help docs:
    • an 'unwrapped' C routine;
    • the modifications needed to incorporate the C code into Cicada;
    • and the accompanying NN.cicada script (which the tutorial goes through the process of debugging);
